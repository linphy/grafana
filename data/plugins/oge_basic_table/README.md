# 基础表格

基础表格插件能接收普通数据源和JOSN数据源，并且实现所有列的显隐，手动编辑单位，自定义报警以及Level报警。

## 使用方法


### Text setting

可以设置`字体样式`，与值的`精度`。


### Column setting

可以设置每一列的`标题名称`与该列`是否显示`。


### Alert setting

可以设置是否启用自定义报警，` 当启用自定义报警时  ，Alert Config中的报警设置将失效`。


### Value setting

可以设置每个标签的`单位``阀值`以及`链接`，当Alert setting中的颜色设置不为警用时，阀值设置项将会显示。

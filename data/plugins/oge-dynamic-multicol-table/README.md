## 使用方法


### Table setting

可以设置表格的`行数`和`列数`，与值的`精度`。
当值为历史值时，可以在值类型中选择要显示值最小值最大值等。


### Cell setting

可以设置每一列的`标题名称`和该列的`宽度`。
可以设置每个单元格的`值`和`值的字体大小`

### Url setting

可以设置每个单元格包括标题的`链接`。

### Alert setting

在Alert Config页面中可以进行报警设置，由于表格为多列表格，可能一行中存在多个报警level，所以无法支持报警。

# 中国省份地图

## 使用方法


### 地图基本设置

可以设置`面板背景色`以及`地图上的所有颜色区域`。

### 高亮设置

高亮是鼠标划过时显示的颜色或状态。

### 数据点

手动添加地图上需要显示的数据点，名称和经纬度是必需项。
